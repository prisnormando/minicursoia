Este repositório contém o material completo para a realização de uma oficina introdutória de 6 horas sobre a aplicação de Inteligência Artificial Generativa (especificamente o Microsoft Copilot) na organização de congressos técnico-científicos na área de vigilância em saúde.

O material foi desenvolvido para ser utilizado por profissionais, pesquisadores e organizadores de eventos acadêmicos, focando na automação de processos, gestão de convidados e estruturação de eventos científicos.

## 📋 Estrutura do Repositório

O repositório está organizado nas seguintes pastas:

*   **`/docs`**: Contém o roteiro completo da oficina, detalhando a carga horária, os módulos teóricos e práticos, e as questões para debate.
*   **`/prompts`**: Coleção de prompts estruturados, prontos para copiar e colar no Microsoft Copilot, organizados por etapa da organização do evento (curadoria, programação, logística, etc.).
*   **`/guias`**: Documentação complementar e guias de uso para facilitar a aplicação prática durante a oficina.
*   **`/skills`**: Configurações avançadas de agentes e *system prompts* para automatizar tarefas específicas.

## 🎯 Objetivos da Oficina

A oficina tem como objetivos principais:
1.  Apresentar os fundamentos dos sistemas algorítmicos e da IA Generativa [1].
2.  Discutir os aspectos éticos e de governança do uso de IA na saúde coletiva.
3.  Capacitar os participantes a criar agentes automatizados usando o Microsoft Copilot Educacional [2].
4.  Aplicar, na prática, a IA para resolver desafios reais na organização de um congresso científico (busca de especialistas, cronograma, logística e certificados).

## 🚀 Como Utilizar Este Material

Este material é de código aberto e pode ser adaptado para diferentes contextos. Para utilizar:

1.  **Facilitadores:** Leiam o arquivo `docs/ROTEIRO_OFICINA.md` para entender a dinâmica das 6 horas de curso.
2.  **Participantes:** Utilizem os arquivos da pasta `/prompts` durante as atividades práticas para interagir com o Copilot.
3.  **Pesquisadores:** O material pode ser referenciado e bifurcado (forked) para criar novas oficinas voltadas a outras áreas da saúde.

## 🛠️ Requisitos Técnicos

Para a realização da oficina, os participantes necessitam de:
*   Computador com acesso à internet.
*   Navegador web atualizado.
*   Acesso ao **Microsoft Copilot** (preferencialmente com licença educacional do Teams).
