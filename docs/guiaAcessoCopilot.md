# Guia: Como Acessar o Microsoft Copilot Educacional

Este guia fornece instruções passo a passo para acessar e utilizar o Microsoft Copilot via web, utilizando a licença educacional do Teams.

---

## Pré-Requisitos

Você deve ter:
*   Uma conta de e-mail fornecida pela sua instituição educacional (ex: `seu.nome@universidade.edu.br`).
*   Acesso ao Microsoft Teams (preferencialmente versão web).
*   Um navegador web atualizado (Chrome, Edge, Firefox ou Safari).

---

## Passo 1: Acessar o Copilot via Web

1.  Abra seu navegador e acesse: **https://copilot.microsoft.com**
2.  Você será redirecionado para fazer login. Clique em "Entrar".
3.  Insira seu e-mail institucional (ex: `seu.nome@universidade.edu.br`).
4.  Insira sua senha institucional.
5.  Se solicitado, complete a autenticação de dois fatores (2FA) usando seu telefone ou aplicativo de autenticação.

---

## Passo 2: Verificar o Tipo de Acesso

Após fazer login, você verá a interface do Copilot. Verifique se está em um dos seguintes modos:

*   **Copilot Pro (Recomendado para Oficina):** Oferece mais respostas por dia e acesso a modelos avançados.
*   **Copilot Gratuito:** Oferece funcionalidades básicas, mas com limite de respostas.

Se sua instituição fornece acesso via Teams, você pode também acessar através do aplicativo Teams:
1.  Abra o Microsoft Teams.
2.  Procure por "Copilot" na barra lateral esquerda.
3.  Clique para abrir o chat do Copilot.

---

## Passo 3: Iniciar um Chat

1.  Na página do Copilot, clique em "Nova Conversa" ou "New Chat".
2.  Você verá um campo de texto com a mensagem "Pergunte-me qualquer coisa..." (ou similar em inglês).
3.  Cole o primeiro prompt (Prompt de Inicialização do Agente) do arquivo `prompts/prompts_oficina.md`.
4.  Pressione Enter ou clique no botão de envio.

---

## Passo 4: Usar os Prompts da Oficina

Após inicializar o agente, você pode usar os prompts subsequentes:

1.  Copie o prompt desejado do arquivo `prompts/prompts_oficina.md`.
2.  Cole no campo de texto do Copilot.
3.  Pressione Enter.
4.  Aguarde a resposta (pode levar alguns segundos).
5.  Revise a resposta e, se necessário, refine o prompt ou faça uma pergunta de acompanhamento.

---

## Dicas de Uso

*   **Contexto:** O Copilot mantém o histórico da conversa. Se você inicializar o agente no início, ele "lembrará" do papel ao longo de toda a conversa.
*   **Copiar Respostas:** Use o botão de cópia (ícone de clipe) para copiar a resposta gerada. Isso facilita colar em documentos do Word, Excel ou e-mails.
*   **Exportar:** Algumas versões do Copilot permitem exportar o histórico da conversa como PDF ou TXT.
*   **Privacidade:** Não insira dados pessoais sensíveis (CPF, RG, telefones) nos prompts. Use variáveis como `[DADO_AQUI]`.

---

## Solução de Problemas

| Problema | Solução |
| :--- | :--- |
| "Erro de autenticação" | Verifique se seu e-mail institucional está correto. Tente fazer logout e login novamente. |
| "Acesso negado" | Sua instituição pode não ter licença Copilot. Contate o departamento de TI. |
| "Copilot não responde" | Pode ser um problema de conexão. Recarregue a página (F5) ou tente em outro navegador. |
| "Limite de respostas atingido" | Aguarde 24 horas ou considere fazer upgrade para Copilot Pro. |
